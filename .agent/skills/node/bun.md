# Bun Commands

Use when `bun.lockb` or `bunfig.toml` is present.

## Install and Dependencies

- `bun install`
- `bun add <package>`
- `bun add -d <package>`

## Validation and Build

- `bun run format`
- `bun run lint`
- `bun run typecheck`
- `bun test`
- `bun run build`

## Runtime

- `bun run dev`
- `bun run start`

## CI Note

Keep bun version pinned and aligned with local tooling.
